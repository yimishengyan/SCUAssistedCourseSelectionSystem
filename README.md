🎯 功能详解
🖱️ 鼠标连点功能
作用：自动帮你点击"刷新"按钮

优点：解放双手，避免手抽筋

风险：别设置太快，小心被系统认为是机器人

👀 课程检测功能
作用：监控指定区域，一旦出现目标课程就提醒你

优点：不用一直盯着屏幕，保护视力

原理：OCR文字识别，比人眼还准（有时候）

⚠️ 注意事项
重要提醒：
不要设置过快的点击间隔：建议2秒以上，否则可能被系统检测到异常

只用于个人学习：别拿去做什么奇怪的事情

抢课有风险：本系统不能保证100%抢到课（但能保证你比别人多喝一杯奶茶的时间）

保持网络通畅：系统再牛，网断了也白搭

提前测试：选课前先找个不重要的时间测试一下

常见问题：
Q：为什么检测不到课程？
A：可能是关键词设置不对，或者监控区域没选对

Q：为什么点击位置不准？
A：设置位置时鼠标要稳，不要手抖！

Q：程序报错了怎么办？
A：检查依赖是否安装完整，或者...重启试试？
